Distribuições do Linux

Uma distribuição é um conjunto de programas e utilitários fornecidos com o Kernel para suprir as necessidades de um conjunto de usuários.

Principais distribuições

  • Slackware, Rede Hat, Debian, Suse, Fedora,CentOS, Ubuntu, etc.
  • Timeline das distribuições Linux
  • Distro watch

Slackware

Uma das distribuições mais antigas em desenvolvimento, possui como características a estabilidade, segunraça e interface de texto para configuração.

Características do Slackware

  • É conhecida por ser direcionada para usuários com grande experiência no Linux.

Red Hat

Distribuição voltada para servidores, vendida juntamente com o suporte técnico.

Características do Red Hat

  • Utiliza pacotes RPM

Fedora

Distribuição utilizada para testes do Red Hat, se tornou uma das principais distribuições devido às inovações.

Características do Fedora

  • Grande número de pacotes (devido à ser baseado no Red Hat)
  • Versões para desktop e servidores
  • Gerenciador de pacote YUM

Debian

Distribuição desenvolvida e mantida por voluntários, possui as versões: stable (produção), testing (versões mais atuais, porém não homologadas) e unstable (usada pelos desenvolvedores Debian).

Características do Debian

  • Utiliza pacotes .DEB;
  • Excelente ferramenta de gerenciamento de pacotes: APT;
  • Uma das maiores distribuição Linux, com diversos pacotes de software e suporte à várias arquiteturas de processador;
  • Ciclo de versões muito longo;

Ubuntu

Distribuição baseada no Debian, patrocinada pela Canonical. Possui como proposta a popularização do Linux, oferecendo um sistema de interface amigável, com acessibilidade e internacionalização.

Características do Ubuntu

  • Uma das distribuições Linux mais populares;
  • Amigável para novatos;
  • Lançamento contínuos e versões LTS
  • Perda de compatibilidade/padronização com Debian
  • Ambiente gráfico próprio (Unity);
  • Diversas variantes: Xubunto, Kubunto, Edubunto, ..., XXXbunto;

Suse

Uma das distribuições comerciais mais antigas, possui excelente ferramenta de instalação e configuração, o YaST.

Características do Suse

  • Licença comercial, voltada para grandes organizações;
  • Utiliza o Yast para gerenciar pacotes
  • Pacotes .RPM

OpenSuse

Distribuição baseada no SUSE, porém voltada de código aberto e gratuíto.

Características do OpenSuse

  • Versão de desktop bonita e atraente;
  • Grande repositório de pacotes, já que é baseado no Suse
  • Utiliza o Yast para gerenciar pacotes
  • Pacotes .RPM

Graphical user interface

Gnome

Ambiente gráfico para desktop utilizado por padrão em diversas distribuições. Objetiva a simplicidade e facilidade de uso.

KDE

Ambiente gráfico para desktop utilizado por padrão em diversas distribuições. Objetiva a modernidade.

Terminal Linux

Ao contrário de ambientes gráficos, onde as operações são realizadas com o ponteiro do mouse, no terminal os comandos são digitados para realizar diversas operações no sistema operacional.

Terminal Linux

  • Na maioria dos sistemas Linux o bash é quem interpreta estes comandos.
  • Como é uma camada acima do S.O, é referenciado como shell.

Por que usar o terminal?

  • Diversas ferramentas e programas disponíveis somente neste modo;
  • Acessar máquinas remotas rapidamente;
  • Os programas permitem grande quantidade de opções para execução
