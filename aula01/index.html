O administrador de redes

Trabalha com tecnologia e com usuários

  • Não deve haver benefícios para usuários específicos, priorisar a comunidade é o objetivo principal


Lida com grande quantidade de recursos

  • Hardware, software, suporte ao usuário, diagnóstico, manutenção e prevenção.

Componentes de uma rede

  • Estações de trabalho de usuário (computadores, notebooks, dispositivos móveis, etc.);
  • Meios de comunicação;
  • Equipamentos de rede que compõem a infraestrutura.

Admin. de Redes vs Admin. de Sistemas

Não são totalmente dissociados, mas existe uma separação entre as funções que cada um executa.

Admin. de Redes vs Admin. de Sistemas

No mundo dos PC, o administrador de redes lida com tudo o que estiver ligado à rede, incluindo os computadores, mas numa definição mais precisa...

Um administrador de redes lida com o gerenciamento de dispositivos de infraestrutura de rede.

Perfil do Administrador

  • O Administrador necessita paciencia, conhecimento e experiência para lidar com os problemas contidianos.
  • Soluções para problemas parecidos nem sempre se aplicam para todos os casos.
  • No passado o administração de sistemas era feita por profissionais não especializados que tinham interesse por tecnologia (curiosos).

O trabalho do administrador de sistemas...

  • Plenejar uma rede eficiente para usuários trabalharem;
  • Implementar serviços e dispositivos;
  • Monitorar o funcionamento;
  • Detectar e corrigir problemas;
  • Proteger de ataques externos e internos;
  • Escalar de acordo com a necessidade;
  • Auditar.

Bugs...

O administrador deve atualizar os sistemas para as versões mais estáveis.
